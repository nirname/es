# Elasticsearch in Docker

Выбрать нужную версию:

```
cd es-2.4.3
```

Поднять кластер из двух контейнеров (по умолчанию, то же самое что `make`):

```
make cluster
```

Или поднять один контейнер:

```
make node
```

Проверить:

```
make check
```

Остановить:

```
make stop
```

Если что-то пошло не так, стереть всё:

```
make refresh
```
